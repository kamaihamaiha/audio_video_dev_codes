## 1.4 编解码工具 ffmpeg

ffmpeg 是 FFmpeg 源代码编译后生成的一个可执行程序，可以作为命令行工具使用。

一个简单的例子：
```shell
ffmpeg -i input.mp4 output.avi
```
该命令做了如下工作：
1. 获取输入源 input.mp4
2. 转码
3. 输出文件 output.mp4

---
### ffmpeg 主要工作流程

1. 解封装(Demuxing)
2. 解码
3. 编码
4. 封装(Muxing)

需要经过 6 个步骤（前后加上输入、输出） =>
1. 读取输入源
2. 音视频解封装
3. 解码每一帧音视频数据
4. 编码每一帧音视频数据
5. 音视频重新封装
6. 输出到目标
