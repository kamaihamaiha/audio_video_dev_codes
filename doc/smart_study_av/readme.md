### 突破口

FFmpeg 是音视频平台及音视频系统开发工作中必不可少的一个组件库，也是我们掌握音视频编解码基础知识与流程的重要抓手。如果能够掌握 FFmpeg 的用法，那音视频方面的一些基本操作都难不住你。
