## Cygwin 

- 直接安装使用。
- 做了一个从 Linux API 到 Windows API 的转换。运行的是 Linux 程序，调用的是 Windows API 接口。
- Cygwin 编译的程序必须要挂载一个 Cygwin.dll 库，否则无法运行。
- 运行速度比原生的 Windows 程序慢一些

---

### Cygwin 环境搭建

- 安装 Cygwin
- 安装 apt-cyg
